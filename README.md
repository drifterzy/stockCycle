### 代码目录 
   本代码为加速程序运行速度，减少网络IO。通过单次访问，将数据以本地的方式保存下来，后续操作将通过直接读取本地文件完成。包括以下：
   * tradeDay目录：保存所有交易日
   * stockPrice目录：保存当期股票上市以来的交易日期，开盘价，收盘价
   * signal目录：保存交易信号
   * result目录：保存交易结果
   * raspberry目录：保存树莓派信号脚本
   * utils目录：保存日期和数据函数
### 调用方法
   运行main函数，输入：
   * 股票代码
   * 起止日期
   * 交易间隔
   * 投入总价
